<h1>Weather</h1>

An application that allows you to delete, edit, and add cities to view the weather. It is also possible to change the language from Spanish to English.
<h3>App include:</h3>
    <li>Flutter</li>
    <li>Animation</li>
     <li>i10n</li>
     <li>openweathermap API</li>
